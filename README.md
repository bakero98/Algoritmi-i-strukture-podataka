**3. treci**

Euklidov algoritam za odredjivanje NZD dva broja.

Neka je sa gcd(m, n) označen NZD brojeva m i n:
Ako je m % n jednako 0, onda je gcd (m, n) jednako n.
U suprotnom, gcd(m, n) je jednako gcd(n, m % n).
